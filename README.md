# C++ Projects

A collection of C++ applications focusing on Control Flow and If-Condition logic.

## Projects

| # | Project | Description |
|---|---------|-------------|
| 01 | Even / Odd Checker | Checks if a number is even or odd |
| 02 | Greatest Number | Finds the greatest among 3 numbers |
| 03 | User Rank Checker | Returns user rank based on points |
| 04 | Simple Calculator | Supports +, -, *, / operations |

## Technologies
- Language: C++
- Concepts: Control Flow, If-Conditions, User Input
